## 📚 Why I Built This

This "Postman clone" project was **not** built to solve a specific real‑world problem or issue.
I created it purely as a way to better understand:

- HTTP APIs and how requests/responses flow through a system  
- FastAPI as a backend framework and how to design clean API layers  
- Streamlit as a quick way to build an API client UI  
- The overall architecture of an API testing tool (similar to Postman)

These files are meant for **learning and experimentation**, not as a production‑ready tool.
